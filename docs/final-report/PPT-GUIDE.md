#### Slide 5: TLS/HTTPS의 한계
**레이아웃**: 제목 + 2단 비교 다이어그램
**내용**:
- 제목: "현재 보안의 심각한 한계"

**왼쪽 (TLS - 문제)**:
```
Agent A ─[TLS 1]→ Gateway ─[TLS 2]→ Agent B
                    ↓
                  복호화
                 ⚠️  평문 노출
                메시지 변조 가능
```

**오른쪽 (SAGE - 해결)**:
```
Agent A ─[TLS + SAGE]→ Gateway ─[TLS + SAGE]→ Agent B
                        ↓
                     암호화 유지
                    ✅ 변조 불가
                   서명 검증으로 차단
```

**강조 박스**:
- "TLS는 구간별 암호화 (Hop-by-Hop)"
- "종단간 무결성 보장 안 됨"
- "SAGE는 종단간 보안 (End-to-End)"
